# HTTP

HTTP -> Protokol untuk komunikasi data dari client (browser) ke server (backend) -> Data berupa JSON.

Method -> GET, POST, PUT, PATCH, DELETE, OPTION

Path -> Alamat tujuan -> example : /index.html
